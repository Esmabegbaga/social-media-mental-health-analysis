 Social Media & Mental Health Analysis

Project Overview

This project explores the relationship between social media usage and mental health indicators using a 2026 dataset.

⸻

 Key Question

Does increased screen time negatively impact mental health?

⸻

 Findings

* A moderate negative correlation was found between screen time and low mood scores
* Higher screen time did not strongly correlate with worse mental health
* Night-time usage showed mixed effects

⸻
 Key Insight
The relationship between social media and mental health is more complex than commonly assumed.

⸻

 Tools Used

* Python
* Pandas
* Seaborn
* Matplotlib
